# Liuli Society of America — Luxury Foundation / Donor-Facing Website

A second premium website version for Liuli Society of America.

## Design direction
- luxury foundation style
- donor-facing visual language
- refined nonprofit storytelling
- elegant dark-gold palette
- full-screen hero and wide banners
- GitHub-ready React + Vite structure

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
